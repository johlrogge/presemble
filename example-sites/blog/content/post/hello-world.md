# Hello, World: Your First Post

Welcome to your new blog. This post is a starting point you can edit or replace with your own content.

## Getting Started

Open the `content/post/` directory and add a new markdown file for each post. Each file follows the schema defined in `schemas/post/item.md`.

## What to Edit

Replace this file with your own story. Update the title, summary, author link, and body to match your content.
